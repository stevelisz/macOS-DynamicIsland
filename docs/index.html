Dynamic Toolbox

macOS productivity suite accessible through notch interaction. Features AI assistant, system monitoring, timers, and development utilities.

To Know

  • macOS 15.1+ (Sequoia)
  • Apple Silicon Mac, not sure if this can ran on a Intel Mac.
  • This is made because I am not sure why my laptop has a notch. So I thought to put some quick tools in there.
  • Mac without a notch can also use this, just click on the middle of your menu bar.
  • I didn't pay for an Apple Developer account so you have to allow the app in security&privacy settings.

Feature Set

AI Assistant

Local AI chat with web search capabilities

Local AI processing via Ollama integration. Supports code analysis, text processing, file operations, and web search functionality.

Requires: Ollama runtime, compatible models (llama3.2:3b recommended)

System Monitor

Real-time CPU, memory, and disk usage tracking

Real-time hardware metrics visualization. CPU core usage, GPU utilization, memory statistics, and storage monitoring.

Implementation: IOKit APIs, host_statistics64, mach kernel interfaces

Clipboard Manager

Smart clipboard history with search and pinning

Persistent clipboard history with search and pin functionality. Supports text, images, and file references.

Storage: UserDefaults, NSPasteboard monitoring

App Launcher

Drag & drop app shortcuts for quick access

Custom application shortcuts via drag-and-drop configuration. Quick access to frequently used applications.

Integration: NSWorkspace, LaunchServices

Timer Functions

Pomodoro timer with session tracking

Pomodoro technique implementation with session tracking and background operation support.

Features: Configurable intervals, statistics, notification system

Weather Data

Location-based weather with beautiful design

Location-based weather information with automatic geolocation detection.

API: OpenWeatherMap, CoreLocation services

Unit Converter

Multi-category converter with live currency rates

Multi-category unit conversion including live currency exchange rates via API integration.

Categories: Length, weight, temperature, currency

Developer Tools

JSON formatting, Base64, and hash utilities

JSON formatting/validation, Base64 encoding, hash generation (MD5, SHA family), and file hash calculation.

Crypto: CommonCrypto, CryptoKit frameworks

Installation

  1. Download DMG from releases page
  2. Mount and drag to Applications folder
  3. Launch app, click notch area to activate
  4. Optional: Install ollama for AI features
