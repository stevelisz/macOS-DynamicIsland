Dynamic Toolbox v2.0.0 - Installation Guide
============================================

INSTALLATION PROCEDURE
======================

1. Download Dynamic-Toolbox-v2.0.0.dmg (~3.1MB)
2. Mount DMG: open Dynamic-Toolbox-v2.0.0.dmg
3. Copy application: cp "Dynamic Toolbox.app" /Applications/
4. Unmount DMG: diskutil unmount /Volumes/Dynamic\ Toolbox
5. Launch: open /Applications/Dynamic\ Toolbox.app
6. Access interface: Click macOS notch area (camera region)

SYSTEM REQUIREMENTS
==================

Minimum:
- macOS 15.1 (Sequoia, build 24B83)
- Architecture: x86_64 or arm64
- Storage: 50MB application bundle
- Memory: 128MB runtime (estimated)

Network dependencies:
- Weather API (optional)
- Currency exchange rates (optional)
- Web search functionality (optional)

AI SUBSYSTEM SETUP (OPTIONAL)
=============================

Prerequisites:
- Ollama runtime environment
- 2GB+ storage for language models
- Local inference processing capability

Installation:
$ curl -fsSL https://ollama.ai/install.sh | sh

Model deployment:
$ ollama pull llama3.2:3b     # Recommended (2GB)
$ ollama pull codellama:7b    # Code-focused (3.8GB)
$ ollama pull llama3.2:7b     # Advanced (7GB)

Verification:
$ ollama list
$ ollama ps                   # Check running models

Dynamic Toolbox will auto-detect running Ollama instance.

TROUBLESHOOTING
==============

Application Launch Issues:
- Verify macOS version: sw_vers
- Check Gatekeeper: spctl -a -vv "/Applications/Dynamic Toolbox.app"
- Console logs: log show --predicate 'process == "Dynamic Toolbox"'

Interface Not Visible:
- Target notch area (top center of display)
- Check for fullscreen applications blocking access
- Application restart: killall "Dynamic Toolbox"; open -a "Dynamic Toolbox"

AI Subsystem Issues:
- Verify Ollama installation: which ollama
- Check service status: ollama ps
- Test model: ollama run llama3.2:3b
- Restart both services if needed

Performance Issues:
- Monitor resource usage: Activity Monitor
- Consider smaller AI models for lower-spec hardware
- Close unused application modules

SECURITY CONSIDERATIONS
======================

Application Security:
- Standard macOS app sandbox
- Local processing for AI operations
- No telemetry or data collection
- Open source codebase for review

Network Communications:
- Weather: OpenWeatherMap API (location data)
- Currency: Exchange rate API (no personal data)
- Web search: DuckDuckGo (privacy-focused)
- AI: Local inference only, no cloud processing

Code Signing:
- Ad-hoc signature for local development
- Not notarized (manual distribution)
- Verify integrity: codesign -dv --verbose=4 "Dynamic Toolbox.app"

UNINSTALLATION
==============

Remove application:
$ rm -rf "/Applications/Dynamic Toolbox.app"

Clean preferences:
$ rm -rf ~/Library/Preferences/fpt.DynamicIsland.*
$ rm -rf ~/Library/Application\ Support/Dynamic\ Toolbox/

Remove Ollama (if installed):
$ /usr/local/bin/ollama --help uninstall
